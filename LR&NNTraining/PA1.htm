Ans: Basically, we use forward propagation and backward propagation for both of the classifiers. The ideas will be described below.

 

For the compute cost function, in Logistic Regression, we did the forward propagation to find the softmax_scores and see if y is 1 or 0 to determine the one_hot_y. Then, calculating the mean of cost by using the formula of cost function. For the compute cost function for Neural Network, we first did the dot product of inputs and the weight from input layer to hidden layer, then doing the tanh of it to get the activation function. Then, did the dot product of activation function and weight from hidden layer to the output layer. Then calculating the softmax_scores. That's basically the forward propagation. Lastly, calculating the mean of cost. We did dot production twice because we have 3 layers, input layer, hidden layer and output layer.

For both of the Logistic Regression and Neural Networks, our fit functions firstly use the forward propagation same as the cost function. Then, we use backward propagration to find the gradient of theta and bias. Lastly, update our theta and bias with different learning rate. The only differences are that NN has two bias and weights to update since it has an input layer and an output layer, but LG only has one input layer.
